# Dolomeets

Dolomeets is a mobile application that helps people discover and join local events by using an intuitive and map-based interface. The app displays real-time event pins across the map of the city of Bolzano, allowing users to tap on an event, view essential information and join with a single click. Dolomeets aims to make event discovery immediate, social and personalized to individual interests. 

The target users are university students, young adults and residents who want to participate more actively in Bolzano’s social and cultural life. Lots of users currently rely on different single sources, like social media pages, posters or word of mouth, making it difficult to know what is happening nearby.
The secondary users include the event organizers, like universities, associations, venues and local businesses) and tourists that are looking for nearby activities during their stay.

The main problem addressed by the project is the fragmentation of event information across multiple channels, such as social media, websites, posters and word of mouth. As a result, users often miss interesting events, spend excessive time searching for information or are unaware of activities happening close to them in real time. In addition, existing solutions rarely integrate social elements, making it difficult for users to see who else is attending or to feel part of a local community. Dolomeets solves this problem by offering a single, location-based solution that simplifies discovery and encourages participation. 

By the end of the course, the expected result of the project is a clearly defined digital product concept supported by a structured prototype or mockup and a detailed project plan. The project aims to demonstrate the application of agile and user-centered design principles, focusing on feature prioritization, user flows and overall system structure rather than on full technical implementation. The final outcome will provide a solid foundation for potential future development.

The project focuses on three highest-priority capabilities. First, the app provides an interactive map of Bolzano that displays events as pins in real time, allowing users to visually explore nearby activities. Second, it enables one-click event participation, making it easy for users to join events, save them to a personal agenda and receive notifications. Third, the app offers personalized event suggestions based on user interests, location and social signals, enhancing relevance and user engagement.

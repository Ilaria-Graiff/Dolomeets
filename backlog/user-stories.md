USER STORIES AND ACCEPTANCE CRITERIA
| Story ID | User Story | Priority | Story Points | Acceptance Criteria |
|---------|-------------|----------|--------------|---------------------|
| 1 | As a student in Bolzano, I want to view a map with event pins, so that I can see what is happening near me in real-time. | High | 5 | Given the user is on the main map screen, When there are active events in Bolzano, Then pins should appear at their precise geographic locations. |
| 2 | As a user, I want to click on an event pin, so that I can see the event title, time, and description. | High | 3 | Given a map with event pins, When the user taps a specific pin, Then an information card pops up showing essential details. |
| 3 | As a student, I want to join an event with one click, so that I can easily participate in social activities. | High | 3 | Given an event detail card is open, When the user clicks the "Join" button, Then the user's status is updated to "Attending." |
| 4 | As a unibz student, I want to log in using my university credentials, so that I can access student-exclusive events safely. | Medium | 8 | Given the login screen, When the user enters a valid @unibz.it email, Then the system authenticates the user and grants access. |
| 5 | As a user, I want to view my history of past events, so that I can keep track of the activities I have joined. | Low | 5 | Given the user profile section, When the user selects "History," Then a list of all previously joined events is displayed. |
| 6 | As a user, I want to filter events by category (e.g., Sports, Culture), so that I only see pins relevant to my interests. | Medium | 5 | Given the map is populated with pins, When the user selects the "Sports" filter, Then only sports-related pins remain visible. |
| 7 | As a student, I want to receive a notification 30 minutes before a joined event, so that I don’t forget to attend. | Medium | 3 | Given the user has joined an event, When it is 30 minutes before the start, Then the system sends a push notification reminder. |
| 8 | As a user, I want to see how many people are already attending an event, so that I can gauge the popularity of the activity. | Medium | 2 | Given the event detail card, When the card is opened, Then a "Current Attendees" count is displayed and updated. |
| 9 | As an organizer, I want to create a new event pin by long-pressing on the map, so that I can quickly broadcast a local activity. | High | 8 | Given the user has organizer permissions, When they long-press the map, Then a form appears to enter event details. |
| 10 | As a user, I want to receive "Smart Suggestions" on my home screen, so that I can discover events tailored to my past interests. | Low | 13 | Given the user has joined 3+ events, When they open the app, Then a "Recommended for You" section highlights matching categories. |
